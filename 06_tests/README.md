# 06_tests

Здесь лежат тестовые скрипты для проверки логики до подключения реального железа.

Запуск из корня проекта:

```bash
python 06_tests/test_01_kinematics.py
python 06_tests/test_02_serial_controller_mock.py
python 06_tests/test_03_sip_analyzer.py
python 06_tests/test_04_safety_logic.py
python 06_tests/test_05_full_scenario_mock.py
```

Эти тесты не требуют подключенного манипулятора.
