# BOM Polkan

Сюда перенести смету и список докупки. Рекомендуемый формат: XLSX + Markdown-резюме.

Критичные докупки:
- микроконтроллер Raspberry Pi Pico / Arduino / STM32;
- PCA9685;
- концевики;
- E-Stop;
- предохранитель 30–40A;
- отдельные DC-DC для Raspberry Pi и сервоприводов;
- клеммы, наконечники, силовые и сигнальные провода.
