# Схема питания «Сегмент-Полкан»

```text
Аккумулятор LiFePO4 24V
   ↓
BMS
   ↓
Предохранитель 30–40A
   ↓
Главный выключатель 24V
   ↓
Распределительная шина 24V / GND
   ├── 24V → DM865C → шаговые двигатели
   ├── 24V → DC-DC 5.1V 5A → Raspberry Pi 5
   ├── 24V → DC-DC 6V 5–10A → сервоприводы / PCA9685
   └── 24V → DC-DC 12V → камера/доп. модули при необходимости
```

Общий GND логики, драйверов и микроконтроллера обязателен для корректной передачи сигналов STEP/DIR/PWM.
